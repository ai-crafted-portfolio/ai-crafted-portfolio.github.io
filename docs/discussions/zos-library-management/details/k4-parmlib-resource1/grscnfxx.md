# GRSCNFxx

**GRS 構成 (STAR / RING)**

*§K-4. PARMLIB 標準メンバ網羅 (4)  (3/10)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | Global Resource Serialization の構成 — STAR (Coupling Facility 経由), RING (CTC リンク経由), NONE (単独) |
| **主要パラメータ** | GRS STAR / GRS RING / GRSDEF DEFAULT — モード選択と Coupling Facility 名 |
| **影響範囲** | IPL 必須・動的反映不可 |
| **関連メンバ** | GRSRNLxx, COUPLExx |
| **注意点** | STAR モードはシスプレックス全体の同意が必要。1 台だけ違うモードにすると GRS 結合不能 |
