# IEACMD00

**システム自動コマンド (固定名)**

*§K-2. PARMLIB 標準メンバ網羅 (2)  (4/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | IBM 提供の IPL 自動コマンドメンバ。COMMNDxx と異なり IPL 段階の早い時点で実行される |
| **主要パラメータ** | T R=00,L=00,SCH=00 (内容は IBM 提供の標準セット) |
| **影響範囲** | IPL 時のみ実行。固定名のため SET 不可 |
| **関連メンバ** | COMMNDxx (利用者カスタム自動コマンド) |
| **注意点** | IBM 提供メンバ。サイトが変更すると IBM 保守 (PTF) で上書きされる。原則変更しない |
