# ai-crafted-portfolio

Claude を活用して作成した検討資料・製品技術情報を公開するポートフォリオサイトです。

## カテゴリ

[**検討資料生成**](discussions/index.md) — 公式マニュアルを Claude が検索できる知識ベース（MCP サーバ）として整備し、検索結果を Excel で構造化、その構造化結果に対して人間が前提条件（SSOT）を指示してカスタマイズし、生成した検討資料の公開

[**製品技術情報**](research/index.md) — 公式マニュアルは多数のページに分割された HTML 形式や英語のみで提供されているものが多く、必要な情報を集めるのに時間がかかります。本サイトでは、その中からユーザが必要とする可能性の高い情報を整理して掲載しています。

## サイトポリシー

本サイトは個人が運営する技術ポートフォリオです。Claude を活用し、公式マニュアル等の公開情報のみを根拠として生成・整理した内容を掲載しています。業務上の固有情報や秘密情報は一切含みません。AI 生成と人手の併用で作成しているため、情報の正確性は保証しません。実装・適用前には公式情報での再確認を推奨します。
