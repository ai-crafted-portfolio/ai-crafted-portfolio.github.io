# OAMSPxx

**Object Access Method (OAM) 設定**

*§K-6. PARMLIB 標準メンバ網羅 (6)  (13/13)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | OAM (DFSMSdfp の Object Storage Manager) の構成 |
| **主要パラメータ** | OSMC THRESHOLD HIGH(100) LOW(50) — Object Storage しきい値 |
| **影響範囲** | SET OAM=xx で動的反映可 |
| **関連メンバ** | IGDSMSxx |
| **注意点** | OAM はオブジェクトストレージ利用サイトのみ。一般的なバッチ運用では未使用 |
