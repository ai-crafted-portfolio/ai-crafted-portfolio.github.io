# LICEFLxx

**License manager**

*§K-3. PARMLIB 標準メンバ網羅 (3)  (3/7)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | z/OS feature のライセンス制御 (一部の追加機能の利用許可・制限) |
| **主要パラメータ** | LICENSE FEATURE('xxx') TERM(...) — feature 別ライセンスの宣言 |
| **影響範囲** | SET LICEFL=xx で動的反映可 |
| **関連メンバ** | IFAPRDxx |
| **注意点** | IFAPRDxx と用途が重なる部分があり、IBM 推奨は IFAPRDxx。サイトによって使い分け |
