# アプリ用ライブラリ — テキスト (T31.7)

**コピーブック・定型 SQL 等**

*§K-12. 運用資材  (7/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | アプリ用テキストライブラリ (COPYLIB, SQLLIB)。コピーブック (COBOL/PL1)、定型 SQL、INCLUDE スクリプト |
| **主要パラメータ** | COBOL の 01 PIC X(...), PL/I の DCL 1 ..., SQL CREATE TABLE 等 |
| **影響範囲** | アプリ業務全般 |
| **関連メンバ** | SCM (アプリソース管理) |
| **注意点** | **原則 SCM 移管** (T58)。本ツールでは移管完了までの一時管理に留める。アプリチームの範疇に入る |
