# IEADMPxx

**SVC ダンプ オプション**

*§K-5. PARMLIB 標準メンバ網羅 (5)  (3/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | SVC ダンプ取得時のデフォルト範囲指定 (RGNCEILING、SDATA 等) |
| **主要パラメータ** | SDATA=(ALLNUC,LSQA,RGN,SUM,SWA,TRT) / Q=YES / TYPE=DEFAULT |
| **影響範囲** | SET DUMP=xx で動的反映可 |
| **関連メンバ** | IEADMR00, IEADMCxx |
| **注意点** | ダンプ範囲広すぎでディスク逼迫、狭すぎで解析不能 |
