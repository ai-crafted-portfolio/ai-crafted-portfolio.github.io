# OMPROUTE

**OSPF/RIP 動的ルーティング**

*§K-9. PROCLIB 標準 PROC 網羅 (2)  (4/12)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | OMPROUTE (動的ルーティング daemon)。OSPF/RIP プロトコルでルート学習 |
| **主要パラメータ** | PARM='-T1 OMP00' / OMPROUTE 設定ファイル |
| **影響範囲** | P OMPROUTE で停止 (= 静的ルートのみ) |
| **関連メンバ** | TCPIP |
| **注意点** | ルート設計サイトのみ。L3 装置がスタティック前提なら起動不要 |
