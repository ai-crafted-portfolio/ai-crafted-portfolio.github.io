# INETD

**Internet daemon**

*§K-9. PROCLIB 標準 PROC 網羅 (2)  (2/12)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | USS 配下の inetd プロセスを起動。telnet/login 等の旧 daemon 集約 |
| **主要パラメータ** | (USS シェル経由起動) |
| **影響範囲** | P INETD で停止可 |
| **関連メンバ** | (USS 関連 — 本ツール対象外) |
| **注意点** | **USS 不在環境では INETD は不要** |
