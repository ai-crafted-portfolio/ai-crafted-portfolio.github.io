# DIAGxx

**Diagnostic ダンプ・トレース設定**

*§K-4. PARMLIB 標準メンバ網羅 (4)  (6/10)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | CTRACE (Component Trace) の自動起動指定、システム ダンプ オプション |
| **主要パラメータ** | CTRACE PARM(ALL) ON / VSM TRACKING(ON) — CTRACE と VSM トラッキング |
| **影響範囲** | SET DIAG=xx で動的反映可 |
| **関連メンバ** | IEADMPxx, IEADMR00, CTncccxx |
| **注意点** | トレース過剰でパフォーマンス低下、過少で解析不能。サイトの障害解析プロセスとセット |
