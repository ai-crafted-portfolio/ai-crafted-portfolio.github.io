# 検討資料生成

公式マニュアルを Claude が検索できる知識ベース（MCP サーバ）として整備し、検索結果を Excel で構造化、その構造化結果に対して人間が前提条件（SSOT）を指示してカスタマイズし、生成した検討資料を公開しています。
製品技術情報が「公式マニュアルから抽出した事実」であるのに対し、こちらは「事実をもとに導いた判断」（技術選定・設計方針・代替案比較など、考えた過程を残す資料）を扱います。
