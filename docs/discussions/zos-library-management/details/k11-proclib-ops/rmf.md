# RMF

**Resource Measurement Facility**

*§K-11. PROCLIB 標準 PROC 網羅 (4)  (6/14)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | RMF (Resource Measurement Facility) 主タスク。性能メトリクス収集 |
| **主要パラメータ** | PARM='MEMBER(00)' (ERBRMFxx 参照) |
| **影響範囲** | P RMF で停止 (= 性能データ収集停止) |
| **関連メンバ** | RMFGAT |
| **注意点** | 性能解析の中核。停止すると性能インシデント解析不能 |
