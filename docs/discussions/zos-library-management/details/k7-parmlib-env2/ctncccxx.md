# CTncccxx

**Component Trace 設定**

*§K-7. PARMLIB 標準メンバ網羅 (7)  (3/14)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | コンポーネント別トレース (例: CTIRRMxx for RACF) のオプション |
| **主要パラメータ** | TRACEOPTS ON / OPTIONS('OPTION1','OPTION2') |
| **影響範囲** | TRACE CT,ON,COMP=xxx,PARM=xx で動的反映可 |
| **関連メンバ** | DIAGxx |
| **注意点** | コンポーネントごとに別メンバ。RACF=CTIRRMxx, IOS=CTIIOxx, SMF=CTISMFxx 等 |
