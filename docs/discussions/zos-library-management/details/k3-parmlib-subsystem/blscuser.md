# BLSCUSER

**IPCS デフォルト exit ルーチン**

*§K-3. PARMLIB 標準メンバ網羅 (3)  (7/7)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | IPCS (Interactive Problem Control System) のユーザカスタマイズ exit。ダンプ解析時の独自処理 |
| **主要パラメータ** | IPCS exit ルーチンの登録 (技術的詳細はマニュアル参照) |
| **影響範囲** | 次回 IPCS 起動時に有効 |
| **関連メンバ** | IEABLDxx (DAE) |
| **注意点** | IPCS カスタマイズは現場での問題解析向上に役立つが、IBM 提供の標準で十分なケース多 |
