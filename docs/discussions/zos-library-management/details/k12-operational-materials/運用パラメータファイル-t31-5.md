# 運用パラメータファイル (T31.5)

**運用 PROC から参照される SYSIN 用パラメータ**

*§K-12. 運用資材  (5/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | 運用 PROC や JCL の SYSIN DD で参照されるパラメータファイル |
| **主要パラメータ** | テキスト形式のキー=値ペア、または ユーティリティの制御文 |
| **影響範囲** | 参照側 PROC/JCL が一斉に影響 |
| **関連メンバ** | PROCLIB の運用 PROC, T31.6 SORT/IDCAMS 制御文 |
| **注意点** | PROC 側 DD との参照整合性チェックを本ツールで提供できるか要検討 (T56)。命名衝突時の検出 |
