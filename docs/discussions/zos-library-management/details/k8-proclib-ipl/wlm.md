# WLM

**Workload Manager**

*§K-8. PROCLIB 標準 PROC 網羅 (1)  (6/17)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | WLM (Workload Manager) のシステム自動起動タスク。PARMLIB IEAOPTxx と Couple Dataset から読む |
| **主要パラメータ** | (自動起動、PARM 通常無し) |
| **影響範囲** | 停止しないこと前提 |
| **関連メンバ** | IEAOPTxx, COUPLExx (WLM CDS) |
| **注意点** | WLM は z/OS の中核。停止 = システム停止と同義 |
