*§K-6. PARMLIB 標準メンバ網羅 (6)  (6/13)*

| 項目 | 内容 |
|---|---|
| **制御内容** | LE (Language Environment) のランタイムオプション。COBOL/PL/I/C 共通のデフォルト |
| **主要パラメータ** | CEEDOPT,CEECOPT — システムレベルとリージョンレベルのデフォルト / RPTSTG(ON) / TRAP(ON) |
| **影響範囲** | SET CEE=xx で動的反映可 |
| **関連メンバ** | (LE 利用するアプリ全般) |
| **注意点** | デフォルト変更で既存アプリの動作が変わるリスク。アプリチームと連携必須 |
