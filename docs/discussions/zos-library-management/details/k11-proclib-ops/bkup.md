# BKUP*

**バックアップ用 PROC**

*§K-11. PROCLIB 標準 PROC 網羅 (4)  (1/14)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | サイトカスタム バックアップ PROC (例: BKUPDB, BKUPCAT, BKUPVOL) |
| **主要パラメータ** | (サイト独自 — DFDSS, IDCAMS BACKUP 等) |
| **影響範囲** | 個別ジョブ単位 |
| **関連メンバ** | DFHSM, DFDSS |
| **注意点** | バックアップウィンドウとリカバリ要件 (RPO/RTO) 整合性必須 |
