# TN3270

**Telnet 3270 サーバ**

*§K-9. PROCLIB 標準 PROC 網羅 (2)  (3/12)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | Telnet 3270 接続を受け付ける TCP/IP サーバ。3270 emulator のメインゲートウェイ |
| **主要パラメータ** | PROFILE.TCPIP の TELNETPARMS / TELNETDEVICE / TELNETGLOBALS |
| **影響範囲** | VARY TCPIP,TN3270,...,STOP で停止 |
| **関連メンバ** | TCPIP, VTAM |
| **注意点** | PCOMM/iEMU 経由のメインフレーム接続点。停止すると 3270 接続不能 |
