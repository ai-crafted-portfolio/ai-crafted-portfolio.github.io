# EXITxx

**System Exit 登録 (旧)**

*§K-7. PARMLIB 標準メンバ網羅 (7)  (6/14)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | システム exit ルーチンの登録 (旧形式、PROGxx EXIT への移行が推奨) |
| **主要パラメータ** | EXIT(SYSREXX) ROUTINE(IRX...) |
| **影響範囲** | IPL 必須 (旧) |
| **関連メンバ** | PROGxx EXIT |
| **注意点** | 新規は PROGxx EXIT ADD/DELETE 構文を使う |
