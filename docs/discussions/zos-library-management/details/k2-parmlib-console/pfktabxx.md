# PFKTABxx

**コンソール PFK キーの割当**

*§K-2. PARMLIB 標準メンバ網羅 (2)  (8/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | コンソール F1〜F24 キーに紐付くコマンド文字列を定義 |
| **主要パラメータ** | PFK(1) CMD('D R,L') / PFK(2) CMD('D A,L') |
| **影響範囲** | SET PFK=xx で動的反映可 |
| **関連メンバ** | CONSOLxx |
| **注意点** | サイト独自キー割当。運用標準を統一しないとオペレータ混乱 |
