# §C-T11. 比較 / 差分検出 (7 機能)

Change Tracker (S15) の比較 / 差分検出 7 機能。

---

## 論点

「3 日前の中身は何か」を保ちつつ、技術制約 (3270 経由 / バイナリ取得困難) と整合する選択をする。

## 個別判定

| # | 機能 | 判定 | 実装方法 / 割切理由 |
|---|---|---|---|
| 1 | テキスト diff | 採用 | VBA + Diff ライブラリ |
| 2 | 複数環境比較 (本番/開発) | 採用 | Excel 比較シートで対応 |
| 3 | 任意 2 版比較 | 採用 | Excel 履歴シート任意行同士で比較 |
| 4 | 時系列詳細表示 | 採用 | Excel ピボット |
| 5 | バイナリ diff (ロードモジュール) | 割切 | RECFM=U は人間可読でないため (T23) |
| 6 | ボリューム単位比較 | 割切 | 3270 経由では時間的に困難 (T24) |
| 7 | USS 配下比較 | 割切 | 対象環境に USS が存在しない (SSOT ④) |

!!! success "T11 の判定結果"
    **7 機能のうち 4 機能採用。** バイナリ・ボリューム・USS は技術制約で割切。
