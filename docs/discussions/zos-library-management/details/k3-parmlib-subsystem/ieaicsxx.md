# IEAICSxx

**(旧) Installation Control Specification**

*§K-3. PARMLIB 標準メンバ網羅 (3)  (5/7)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | WLM 移行前の旧 SRM (System Resource Manager) ベースのワークロード分類定義 |
| **主要パラメータ** | SUBSYS=JES,PGN=10 / SUBSYS=TSO,PGN=11 — Performance Group 割当 |
| **影響範囲** | IPL 必須・動的反映不可。WLM 移行で廃止予定 |
| **関連メンバ** | IEAIPSxx (連動) |
| **注意点** | z/OS V2.x 以降は WLM ポリシー (Goal Mode) が標準。IEAICS は使うべきでない |
