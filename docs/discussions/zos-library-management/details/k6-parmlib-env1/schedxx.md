# SCHEDxx

**スケジューラ・PPT (Program Properties Table)**

*§K-6. PARMLIB 標準メンバ網羅 (6)  (4/13)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | JES 起動オプションとプログラム属性テーブル。バッチプログラムのキー、SWAP、DREF 等の属性 |
| **主要パラメータ** | PPT PGMNAME(IEFBR14) NOSWAP / PPT PGMNAME(DSNUTILB) NOSWAP NOPREF |
| **影響範囲** | SET SCH=xx で動的反映可 |
| **関連メンバ** | WLM, IEAOPTxx |
| **注意点** | PPT エントリミスでサブシステムプログラムが正しく動作しない可能性。ベンダー指定エントリは保持 |
