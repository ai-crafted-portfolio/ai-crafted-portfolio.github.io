# 監視・通知定義 (T31.9)

**監視ツール定義 (閾値・宛先・文面)**

*§K-12. 運用資材  (9/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | 監視ツール (NetView, NetCool, Tivoli, Splunk 等) の定義ファイル |
| **主要パラメータ** | 閾値、対象メトリクス、宛先メールアドレス・ページャ番号、文面テンプレ |
| **影響範囲** | **変更で通知が止まる→検知遅延、または過剰通知→alert fatigue。両側のリスク** |
| **関連メンバ** | ALERT*/NOTIF*/HEALTH* PROC (T30.6) |
| **注意点** | **本番反映前にテスト alert を発火させる手順を本ツール上で支援** (T60)。閾値変更時のシミュレーション必須 |
