# 業務 JCL (T31.1)

**業務日次・月次バッチの実行 JCL**

*§K-12. 運用資材  (1/9)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | 業務処理用 JCL (運用部門が日常編集する JOB 定義)。業務ロジックそのものではなく、EXEC 文・DD 文・パラメータ指定が中心 |
| **主要パラメータ** | JOB(...) / EXEC PGM=... / DD DSN=... / SYSIN DD * / 環境変数置換 |
| **影響範囲** | 本番投入で当該業務に影響。日次・月次・年次バッチ全般 |
| **関連メンバ** | SCM (アプリ部門) との境界整理 (T52), 運用パラメータファイル T31.5 |
| **注意点** | **SCM 一次管理が原則**。本ツールは本番反映後の追跡のみとするのが推奨方針 (T52)。直接編集による事故防止のため、レビュー・承認プロセス必須 |
