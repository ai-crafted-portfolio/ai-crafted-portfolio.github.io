# BPXPRMxx

**USS 設定 (本ツール対象外)**

*§K-6. PARMLIB 標準メンバ網羅 (6)  (12/13)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | USS (UNIX System Services) の root ファイルシステム、TIOCMAX、MAXPROCSYS 等 |
| **主要パラメータ** | ROOT FILESYSTEM('OMVS.ROOT.HFS') TYPE(HFS) / MOUNT FILESYSTEM(...) MODE(RDWR) |
| **影響範囲** | SET OMVS=xx で動的反映可 |
| **関連メンバ** | IOEPRMxx (zFS), IKJTSOxx |
| **注意点** | **本ツール対象環境では SSOT ④ により USS 不在のため、本メンバ自体が存在しない** |
