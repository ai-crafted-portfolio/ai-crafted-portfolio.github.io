# IEABLDxx

**DAE (Dump Analysis and Elimination) 設定**

*§K-1. PARMLIB 標準メンバ網羅 (1)  (12/12)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | DAE のコレクション定義。重複ダンプの自動抑止ルール |
| **主要パラメータ** | DAE=START / DAE=STOP / SVCDUMP / SYSMDUMP |
| **影響範囲** | 動的反映可 (SET DAE=xx) |
| **関連メンバ** | DIAGxx, IEAABDxx |
| **注意点** | DAE 抑止が強すぎると診断材料を逃す。逆に弱すぎるとダンプ大量発生でディスク逼迫 |
