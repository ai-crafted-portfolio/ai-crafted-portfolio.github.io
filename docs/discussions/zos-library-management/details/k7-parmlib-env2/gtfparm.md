*§K-7. PARMLIB 標準メンバ網羅 (7)  (7/14)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | GTF (Generalized Trace Facility) の起動オプションと記録対象 |
| **主要パラメータ** | TRACE=SYSM,SLIP,SVC / END |
| **影響範囲** | GTF 起動時に読み込み (S GTF,xx) |
| **関連メンバ** | DIAGxx, CTncccxx |
| **注意点** | GTF はトレース用。本番運用では基本停止、解析時のみ起動 |
