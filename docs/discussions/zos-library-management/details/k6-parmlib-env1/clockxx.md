# CLOCKxx

**システム時刻設定**

*§K-6. PARMLIB 標準メンバ網羅 (6)  (7/13)*

---

| 項目 | 内容 |
|---|---|
| **制御内容** | システム時刻 / タイムゾーン / ETR (External Time Reference) モード |
| **主要パラメータ** | OPERATOR PROMPT / TIMEZONE E.09 (= +9h, JST) / ETRMODE(YES) / STPMODE(YES) |
| **影響範囲** | IPL 必須・動的反映不可 |
| **関連メンバ** | TIMEZONE |
| **注意点** | サマータイム/STP モード変更は社内システム間連携に影響大。日本は通常 JST 固定 |
